What OpenShell Adds

Current: Direct Spawner

Runner │ spawn() ▼ K8s Job / Podman container │ │ (container securityContext only) │ ▼ Sandbox Container POST /v1/agent/run GET /v1/agent/events

OpenShell: Secured Spawner

Runner │ gRPC: CreateSandbox ▼ OpenShell Gateway │ Compute Driver (K8s/Podman/Docker) ▼ Supervisor injected │ Landlock (filesystem isolation) │ seccomp (syscall filtering) │ Network namespace (L4/L7 policy) │ JWT auth (per-sandbox identity) ▼ Sandbox Container HARDENED POST /v1/agent/run SAME GET /v1/agent/events SAME

Deployment Topologies

Podman (RHEL Production)

RHEL Host ├── Temporal Server (container) ├── PostgreSQL (container) ├── Workflow Runner (container) ├── OpenShell Gateway (container, Podman driver) │ └── Podman socket mount (DooD) ├── MCP Servers (containers) └── Sandbox containers (spawned by Gateway via Podman)

Kubernetes (Production)

K8s Cluster ├── Temporal Server (Deployment) ├── PostgreSQL (StatefulSet) ├── Workflow Runner (Deployment) ├── OpenShell Gateway (Deployment, K8s driver) │ └── Sandbox CRD controller ├── MCP Servers (Deployments) └── Sandbox pods (created as Sandbox CRs)
